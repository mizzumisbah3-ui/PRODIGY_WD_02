# ⏱️ PRODIGY_WD_02 - Stopwatch Web Application

## 📌 Overview

This project is a modern and responsive Stopwatch Web Application developed as part of Task 02 of the Prodigy InfoTech Web Development Internship.

The application allows users to accurately measure elapsed time with functionalities such as Start, Pause, Reset, and Lap Recording. It is designed with a clean and user-friendly interface, ensuring a smooth experience across different devices and screen sizes.

---

## 🚀 Features

* Start Stopwatch
* Pause Stopwatch
* Reset Stopwatch
* Lap Time Recording
* Accurate Time Tracking
* Responsive Design
* Modern User Interface
* Smooth Button Interactions
